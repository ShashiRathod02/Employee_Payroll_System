# Employee Payroll System

A Java-based Employee Payroll System that manages employee records and calculates salary details.

## Features

- Add employees to payroll records
- Remove employees by ID
- Search for employees by ID
- Calculate basic salary, gross salary, and net salary
- Display detailed employee salary reports
- Display a formatted payroll summary table

## Project Structure

- `Employee.java` — Employee model with salary calculation methods
- `Payroll.java` — Payroll record manager with add/remove/search/display operations
- `Main.java` — Program entry point and demo workflow
- `README.md` — Project documentation

## How to Run

1. Open a terminal in the project folder
2. Compile the Java files:

```bash
javac *.java
```

3. Run the application:

```bash
java Main
```

## Sample Output

The program prints added employee messages, detailed employee salary information, a payroll summary table, and demonstration of search/remove operations.
